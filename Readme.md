 Student Registration and Login System
This is a simple web application for student registration and login.
It allows users (students) to create accounts, log in, and manage their data using a database.
 Features
- Student registration
- Student login
- Password authentication
- Database storage (MySQL)
- Simple and user-friendly interface
  - Technologies Used
- HTML
- CSS
- PHP ( backend language)
- MySQL Database
-  How to Run
1. Clone the repository
2. Import the database file into MySQL
3. Start XAMPP/WAMP server
4. Run the project in localhost (http://localhost/student_registration_system)
